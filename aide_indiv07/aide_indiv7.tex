\section*{\center{Aide individualisée: Signe de ax+b}}

\subsection*{Faire un tableau de signes}

\begin{enumerate}
  \item Compléter le tableau:
  
 \begin{center}
  $
  \begin{array}{|c|m{8mm}|m{8mm}|m{8mm}|m{8mm}|m{8mm}|m{8mm}|m{8mm}|m{8mm}|m{8mm}|} 
   \hline
   x & -2 & -1 & 0 \ & 1\ & 2 \ & 3 \ & 4 \ & 5 \ & 6\  \\
  \hline
  5x-10 & -20 & \ & \ &\  &\  & \ &\  &\  &\\ 
  \hline
  \end{array}
$
\end{center}
\item Pour quelle valeur de $x$, $5x-10$ vaut 0?
\item Donner deux exemples de valeurs de $x$ pour lesquelles $5x-10$ est
négatif?
\item Donner deux exemples de valeurs de $x$ pour lesquelles $5x-10$ est
positif?

\medskip

D'après les valeurs trouvées, il \ul{semblerait} que $5x-10$ soit négatif pour
$x \ldots \ldots \ldots$, nul pour $x=\ldots$ et positif pour $x \ldots \ldots
\ldots$.

Vérifions-le par le calcul:
\item Résoudre:
$5x-10=0$

$5x-10<0$

$5x-10>0$

\item Compléter:

 Pour $x \in ]- \infty; \ldots [$, $5x-10$ est négatif.
 
Pour $x=\ldots$, $5x-10$ est nul.

 Pour $x \in ]2;+ \infty[$, $5x-10$ est \ldots \ldots \ldots \ldots \ldots.
 
 \enskip
 Schématisons les trois dernières phrases dans un tableau:   
 \begin{center}
 \includegraphics[width=14cm]{images/ex.jpg}
 \end{center}       
\end{enumerate}
 
\subsection*{Lire un tableau de signes}

\begin{center}

$\begin{array}{|c|ccccc|}
\hline
 x & -\infty & \ \ \ \ \ \ \ \ \ \ \ & 1 &\ \ \ \ \  \ \ \ \ \ & +\infty \\
 \hline
 1-x & \ \ & + & \ \ \ \ 0 \ \ \ \ & - & \ \ \\
 \hline
 \end{array}$
\end{center}

\begin{enumerate}
  \item Pour quelle valeur de $x$, $1-x$ vaut $0$?
  \item Donner des exemples de valeurs de $x$ pour lesquelles $1-x$ est négatif.
  \item Pour quelle valeur de $x$, $1-x$ est-il positif? (donner la réponse sous
  forme d'intervalle)
  \item Pour $x=-2$, quel est le signe de $1-x$? Même question pour $x=4$.
\end{enumerate}

\subsection*{A vous de jouer\ldots}

\begin{enumerate}
  \item Faire le tableau de signes de $7x+3$.
  
  \textbf{Méthode:} 
  \begin{itemize}
    \item [$\bullet$] Résoudre $7x+3=0$.
    \item [$\bullet$] Résoudre $7x+3<0$.
    \item [$\bullet$] Compléter le tableau de signes.
  \end{itemize}
  
  \item Faire le tableau de signes de $-4x+5$.
  \item Résoudre $(2x-1)(x+6)<0$.
  
  \textbf{Méthode:}
  \begin{itemize}
    \item [$\bullet$] Résoudre $2x-1=0$.
    \item [$\bullet$] Résoudre $x+6=0$.
    \item [$\bullet$] Mettre les valeurs trouvées dans le tableau (dans la
    ligne ``des $x$'')
    \item [$\bullet$] Etudier le signe de $2x-1$ sur chaque intervalle et
    compléter le tableau.
    \item [$\bullet$] Etudier le signe de $x+6$ sur chaque intervalle et
    compléter le tableau (juste en dessous).
    \item [$\bullet$] Etudier le signe du produit en utilisant la règle des
    signes, puis lire le tableau pour conclure.
  \end{itemize}
\end{enumerate}
