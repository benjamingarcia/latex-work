\textbf{Exercice 8:} Pour chacune des propriétés (P1),(P2) et (P3): Enoncer la réciproque, dire si elle est vraie ou fausse.
  Dans le cas où elle est fausse, donner un contre-exemple
  (cela peut-être une figure).


\enskip

\begin{itemize}
  \item [$\bullet$] (P1): Si le quadrilatère $ABCD$ est un carré alors $(AC)$ et
  $(BD)$ sont perpendiculaires.

 
   \enskip
   
   \item [$\bullet$] (P2): Si le quadrilatère $ABCD$ est un losange alors
   $AB=BC=CD=DA$.
   
   \enskip
   
  \item [$\bullet$] (P3): Si le quadrilatère $ABCD$ est un rectangle alors
  $[AC]$ et $[BD]$ se coupent en leur milieu.
\end{itemize}
