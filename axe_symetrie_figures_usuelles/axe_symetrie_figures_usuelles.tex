\section{Figures symétriques}


\ul{Définition}: Deux figures sont \textbf{symétriques} par rapport à une
droite, si en pliant suivant cette droite, les deux figures se superposent.
Cette droite est l'{axe de symétrie}.


\medskip


\ul{Exemple}:

\begin{center}
\includegraphics[width=15cm]{images/figure_symetrique.jpg} 
\end{center}

\ul{Méthode pour tracer le symétrique dela droite (d) par rapport à la
droite ($\Delta$)}:

\enskip


1. Je place deux points A et B sur la droite (d).

2. Je trace le symétrique du point A, je le note A'.

3. Je trace le symétrique du point B, je le note B'.

4. Je trace la droite (A'B'). (A'B') est la droite symétrique de la droite (d).
\end{minipage}
\end{tabular}
